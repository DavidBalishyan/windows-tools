# Windows Tools Collection

A curated collection of essential utilities for Windows activation and management. This repository aggregates open-source tools to streamline your Windows setup experience.

## üìÇ Included Tools

### 1. [Microsoft Activation Scripts (MAS)](./MAS)

> **Location:** [`/MAS`](./MAS)

A comprehensive activator for Windows and Office.

- **Key Features:**
  - **HWID**: Permanent digital license for Windows 10/11.
  - **Ohook**: Permanent activation for Office.
  - **KMS38**: Activates Windows 10/11 Server till 2038.
  - **Online KMS**: Standard activation method.
- **Documentation:** See [MAS README](./MAS/README.md) for detailed usage instructions.

### 2. [WinRAR Activator](./WinRAR-Activator)

> **Location:** [`/WinRAR-Activator`](./WinRAR-Activator)

A lightweight utility to fully activate WinRAR.

- **Key Features:**
  - Automatically generates and installs `rarreg.key`.
  - Supports both PowerShell (one-liner) and Batch script execution.
  - Works on standard WinRAR installations.
- **Documentation:** See [WinRAR Activator README](./WinRAR-Activator/README.md) for detailed usage instructions.
